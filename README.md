# 4EverCommunitys.Shopify

A simple adaptive kid-learning web app that teaches:
- Colors
- Numbers
- Shapes

## What this app does
- Lets you choose a child name and language.
- Creates quick rounds with visual prompts.
- Uses text-to-speech in the selected language (browser support varies).
- Increases difficulty as the child earns points and streaks correct answers.

## Run locally
Open `index.html` in any modern browser.

No build step required.
